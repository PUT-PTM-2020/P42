# P42
FM radio with temperature sensor.

**Used modules:**
1. Radio module - RDA5807M,
2. Temperature sensor - DS18B20,
3. LCD display 16x2


**Pinout**

LCD display\
VSS <-> GND\
VDD <-> 5V\
V0  <-> potentiometer\
RS  <-> PE0\
RW  <-> GND\
E   <-> PE1\
D0 - D3 - empty\
D4  <-> PD8\
D5  <-> PD9\
D6  <-> PD10\
D7  <-> PD11\
A   <-> 5V\
K   <-> GND
